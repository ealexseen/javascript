In this example, slice() and substring() are used in the same manner and, ironically enough, return the same values. When given just one argument, 3, both methods return “lo world”, as the second “l” in “hello” is in position 3. When given two arguments, 3 and 7, both methods return “lo w” (the “o” in “world” is in position 7, so it is not included). Why have two methods that do the exact same thing? Truthfully, the methods aren’t identical, but they differ only in how they deal with arguments that are negative numbers.
